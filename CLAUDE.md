# DGS IA First — Luciane Baldo | DB1 Global Software

## Sobre este repositório
Exercícios práticos da Certificação DGS IA First.
**Papel:** Delivery Manager Sênior
**Empresa:** DB1 Global Software

## Cenário dos exercícios
Empresa fictícia: **NovaTech** — logística, 1.200 funcionários.
Projeto: assistente de IA com RAG para time de atendimento (45 pessoas).
Meta: reduzir tempo de consulta de 12 min para < 2 min por chamado.

## Estrutura do repositório
| Pasta | Conteúdo |
|-------|----------|
| `context/` | Documentação oficial da NovaTech (fonte de verdade) |
| `exercises/` | Definições dos exercícios (enunciados) |
| `deliverables/dm/` | Entregáveis produzidos — papel Delivery Manager |

## Regras para entregáveis
- Nunca repetir contexto que já existe em `context/` ou `exercises/`
- Referenciar documentos pelo caminho relativo, não copiar conteúdo
- Todo entregável começa com frontmatter YAML
- Usar tabelas para matrizes de risco e cronogramas
- Linguagem: português formal
